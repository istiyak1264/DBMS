# 🍽️ Mess Management System

A complete, full-stack web application for managing mess operations, including member management, meal tracking, expense management, and comprehensive reporting.

## ✨ Features

### Core Features
- **User Authentication**: Secure JWT-based authentication with login/registration
- **Member Management**: Add, edit, delete, and view members with role-based access
- **Meal Tracking**: Record and manage daily meals per member
- **Expense Management**: Track and categorize expenses with payment methods
- **Dashboard**: Real-time statistics, charts, and activity feed
- **Reports**: Generate detailed reports with date filtering and export capabilities

### Technical Features
- **Responsive Design**: Fully responsive mobile-first design
- **Dark Theme**: Modern dark UI with gradient accents
- **Real-time Updates**: Live data updates and notifications
- **Data Export**: CSV/JSON export for meals, expenses, and reports
- **Search & Filter**: Advanced search and filter capabilities
- **Charts & Analytics**: Interactive charts for data visualization
- **Keyboard Shortcuts**: Enhanced productivity with keyboard shortcuts
- **Accessibility**: WCAG compliant with keyboard navigation

## 🏗️ Tech Stack

### Backend
- **Framework**: Flask 2.3.3
- **Database**: MongoDB 6.0
- **Authentication**: JWT with Flask-JWT-Extended
- **ORM/ODM**: PyMongo
- **Security**: bcrypt, CORS
- **Python**: 3.10+

### Frontend
- **Framework**: React 18
- **Build Tool**: Vite 4
- **Styling**: CSS Modules with responsive design
- **Charts**: Chart.js
- **Icons**: React Icons
- **HTTP Client**: Axios
- **Routing**: React Router v6

### DevOps
- **Containerization**: Docker & Docker Compose
- **Reverse Proxy**: Nginx
- **Database Management**: Mongo Express (optional)
- **Caching**: Redis (optional)

## 📋 Prerequisites

- Docker and Docker Compose (Recommended)
- Node.js 18+ and npm (for local development)
- Python 3.10+ (for local development)
- MongoDB 6.0+ (for local development)
